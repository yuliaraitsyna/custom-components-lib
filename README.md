## How to run the app
1. Clone repository and run the ap

```
    git clone https://github.com/yuliaraitsyna/custom-components-lib.git

    npm install

    npm run storybook
```

2. Install the package to your project

```
    npm install @yuliaraitsyna/custom-components-lib@1.0.0
```
